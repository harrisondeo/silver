# Assumptions

- Forms do not need to functionally submit data
- A styling library is not required
- Custom assets are not required
- The application should be responsive
- The application should be accessible
- The application should be performant
- The application should be built using React
- The application should be built using TypeScript
- Validation code for form elements is not required, only the UI of the different validation states
- No consideration is needed for legacy browser support
- No consideration is needed for SEO
- No consideration is needed for internationalization
- No consideration is needed for server side rendering vs client side rendering
